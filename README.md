# 🌐 Portfolio Website

A personal portfolio website built using HTML, CSS, and JavaScript to showcase my skills, projects, and contact information.

## 🔗 Live Demo

https://yourusername.github.io/Portfolio/

## ✨ Features

- Responsive design for desktop and mobile
- About Me section
- Skills section
- Projects with GitHub and Live Demo links
- Contact form
- Smooth scrolling and simple animations
- Clean and modern user interface

## 🛠 Tech Stack

- HTML – Structure of the website
- CSS – Styling and responsive layout
- JavaScript – Interactive features and animations

## 🚀 How It Works

- Open the portfolio website.
- Explore the About, Skills, and Projects sections.
- Click the GitHub button to view project source code.
- Click the Live Demo button to try the projects.
- Use the Contact section to get in touch.
